# Python Voice Assistant

This project is a Python-based voice assistant that listens to voice commands and responds using speech recognition and text-to-speech.

---

## Features

- Wake word detection: "Hey Assistant"
- Speech recognition using microphone
- Voice responses using text-to-speech
- Tells current time and date
- Searches Wikipedia and reads summary
- Opens Google and YouTube using voice commands
- Stops safely on voice command

---

## Libraries Used

- SpeechRecognition (for voice input)
- pyttsx3 (for text-to-speech)
- wikipedia (for information search)
- datetime (for time and date)
- webbrowser (for opening websites)
